Level-of-Detail Preprocessor

The Preprocessor and its source code is available for download on this page. This page also details the recommended system requirements, and usage of the software.
Recommended System
ComponentRecommendation
CPUIntel i5 CPU or better
RAM4 to 8 GB
StorageDue to the number of temporary files generated during preprocessing, we recommend about 5 times the size of the model in free space on the hard drive. For example, processing a 1 GB model requires around 5 GB of free space. We also recommend using a fast storage mechanism such as a 7200rpm harddrive or Solid State Drive but this is not necessary for the operation of this software.
Operating SystemThe software has been tested on both Windows 8.1 64-bit and Ubuntu 12.10 64-bit operating systems.
JREThis software requires a 64-bit Java Runtime Environment of version 1.7 or later.

Installation

  1. Download the zip file containing the executables above.
  2. Unzip the folder into a target directory.
  3. Do not modify the folder structure or move the PLYPreprocessor.jar file.

Usage

  1. Run the PLYPreprocessor.jar file by double clicking on it.
  2. Use the Pick input file button to choose the input model.
  3. Use the Pick output file button to choose the output file.
  4. If the model was recorded in a Z-up orientation, tick the Swap YZ axis box to convert the model into the coordinate space used by the Renderer.
  5. Click the Go button to begin processing.
  6. Watch for any errors in the console box.
  • All temporary files are stored in a temporary directory in the same folder as the chosen output file.
  • The Reset button can be used to clear the fields, output box, and progress bar.
  • The Abort button can be used to stop the preprocessing if it is taking too long. This may require manual removal of temporary files.
